# Privacy Policy / 隐私政策

> Last Updated / 最后更新: 2026-01-31

Thank you for visiting Gin's Blog. We respect your privacy and are committed to protecting your personal data. This privacy policy explains what information we collect and how we use it. / 感谢您访问 Gin 的博客。我们尊重您的隐私并致力于保护您的个人数据。本隐私政策说明了我们收集的信息及其用途。

## 1. Information We Collect / 我们收集的信息

### 1.1 Automatically Collected Data / 自动收集的数据
When you visit our site, we may collect minimal technical data to optimize your experience and ensure security:
当您访问我们的网站时，我们可能会收集最少量的技术数据以优化您的体验并确保安全：
- **Network Info / 网络信息**: IP address, ISP, connection type (e.g., WiFi/4G) for routing optimization. / IP 地址、ISP、连接类型（如 WiFi/4G），用于路由优化。
- **Device Metrics / 设备指标**: Screen resolution, device memory, and CPU concurrency to determine optimal content rendering quality. / 屏幕分辨率、设备内存和 CPU 并发数，用于决定最佳的内容渲染质量。
- **Location Data / 位置数据**: Approximate location (City/Country) derived from IP address for analytics purposes. / 基于 IP 地址的大致位置（城市/国家），用于统计分析。

### 1.2 User Account Data / 用户账户数据
If you choose to log in via third-party OAuth providers (GitHub, Google, Discord), we store only the public information provided by them:
如果您选择通过第三方 OAuth 提供商（GitHub、Google、Discord）登录，我们仅存储其提供的公开信息：
- Username / 用户名
- Public Email Address / 公开电子邮件地址
- Avatar URL / 头像链接
- Third-party User ID / 第三方用户 ID

## 2. How We Use Your Data / 我们如何使用您的数据

- **Performance Optimization / 性能优化**: Device metrics help us serve lighter content to slower connections or lower-end devices. / 设备指标帮助我们为慢速连接或低端设备提供更轻量的内容。
- **Security / 安全防护**: IP addresses are used to detect routing anomalies and prevent abuse. / IP 地址用于检测路由异常并防止滥用。
- **Social Features / 社交功能**: Your account data allows you to post comments and interact with content. / 您的账户数据使您能够发表评论并与内容互动。

**We do NOT share your personal data with advertisers or third parties.** / **我们不会与广告商或第三方共享您的个人数据。**

## 3. Data Storage / 数据存储

Your data is stored securely on **Cloudflare D1 & KV** infrastructure. We employ industry-standard security measures including strict access controls (Zero Trust) and encryption. / 您的数据安全地存储在 **Cloudflare D1 & KV** 基础设施上。我们采取行业标准的安全措施，包括严格的访问控制（零信任）和加密。

## 4. Your Rights / 您的权利

You have full control over your data. Upon request, or by using the "Logout & Clear Session" features in the user interface, you can manage your session data. / 您拥有对自己数据的完全控制权。通过请求或使用用户界面中的“注销并清除会话”功能，您可以管理您的会话数据。

## 5. Third-Party Services / 第三方服务

We use the following services which may process data: / 我们使用以下可能处理数据的服务：
- **Cloudflare**: Hosting, Security, and Edge Computing. / 托管、安全和边缘计算。
- **OAuth Providers**: GitHub, Google, Discord (for authentication only). / 身份验证服务。
